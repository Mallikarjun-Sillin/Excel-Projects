# Global Sales Disruption Project

This Excel project analyzes supply chain and shipping data to identify disruptions affecting delivery performance. The data was cleaned, organized, and visualized using Pivot Tables, Charts, and Slicers to uncover trends in delays, shipping costs, transportation modes, and product categories.

## Key Insights

* Geopolitical conflicts caused the highest delivery delays.
* The Suez route had the highest shipping cost.
* Sea transportation experienced more delays than air transportation.
* Consumer Electronics recorded the highest shipping cost.

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Data Cleaning

## Dashboard Preview

The dashboard provides interactive filtering and visual analysis of supply chain performance and disruption factors.
