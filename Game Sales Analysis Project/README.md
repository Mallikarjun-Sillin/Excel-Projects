# Game Sales Analysis Project

This Excel project explores global video game sales data across different genres, platforms, publishers, and years. The dataset was analyzed using Pivot Tables, Charts, and Slicers to identify sales trends and market performance.

## Key Insights

* Action games generated the highest global sales.
* PS2 was the best-selling gaming platform.
* Nintendo was one of the leading publishers in total sales.
* Global game sales peaked during the 2007–2009 period.

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Data Analysis

## Dashboard Preview

The dashboard allows users to interactively explore game sales trends by genre, platform, publisher, and year.
